# Modul Praktikum Machine Learning 🤖🌾

Selamat datang di repositori **Modul Praktikum Machine Learning**. Modul ini dirancang khusus untuk mahasiswa yang ingin mempelajari implementasi Machine Learning secara praktis, mulai dari dasar algoritma hingga deployment aplikasi cerdas, dengan fokus studi kasus pada **Smart Agriculture**.

## 🚀 Panduan Singkat

Seluruh praktikum dalam modul ini dirancang untuk dijalankan di **[Google Colab](https://colab.research.google.com/)**. Anda tidak perlu melakukan instalasi di komputer lokal.

1.  **Buka File Modul**: Pilih pertemuan yang ingin dipelajari dari daftar di bawah.
2.  **Copy-Paste Kode**: Salin blok kode Python ke dalam cell Google Colab.
3.  **Jalankan Cell**: Tekan `Shift + Enter` untuk mengeksekusi kode.
4.  **Latihan**: Kerjakan bagian "Latihan/Tugas" di akhir setiap modul untuk memperdalam pemahaman.

## 📚 Daftar Modul Pertemuan

Berikut adalah urutan pembelajaran praktikum selama satu semester (minus Kontrak Kuliah, UTS, dan UAS):

### Fase 1: Dasar Machine Learning & Evaluasi
*   **[Modul 02 - Dasar Konsep ML](modul-02-pengenalan-ml.md)**: Klasifikasi (Supervised) vs Clustering (Unsupervised).
*   **[Modul 03 - Algoritma Klasik](modul-03-algoritma-klasik.md)**: Implementasi *Logistic Regression* & *Decision Tree*.
*   **[Modul 04 - Algoritma Lanjut](modul-04-algoritma-lanjut.md)**: Perbandingan performa *SVM*, *Random Forest*, dan *XGBoost*.
*   **[Modul 05 - Evaluasi Model I](modul-05-evaluasi-model-i.md)**: Memahami *Confusion Matrix*, *Precision*, *Recall*, dan *F1-Score*.
